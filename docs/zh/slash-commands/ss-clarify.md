# /ss-clarify

记录需求澄清和设计决策。

## 语法

```
/ss-clarify
```

## AI 行为

执行此命令后，AI 会：

1. 识别当前的歧义点
2. 向你提问以澄清
3. 记录决策到 clarify.md
4. 更新相关 artifacts

## 使用场景

- 需求不清晰时
- 有多种实现方案时
- 需要记录重要决策时

## 示例交互

```
你: /ss-clarify

AI: 我注意到以下几点需要澄清：
    1. 用户登录失败后应该如何处理？
    2. 密码强度要求是什么？

    请逐一回答这些问题。

你: 1. 显示错误信息，3次失败后锁定账户
    2. 至少8位，包含大小写和数字

AI: 已记录到 clarify.md：
    - 登录失败处理：显示错误，3次锁定
    - 密码要求：8位+，大小写+数字
```
