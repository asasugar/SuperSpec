# Tutorials

Master SuperSpec through hands-on tutorials.

## Tutorial List

### Beginner

**[Your First Spec](/tutorials/first-spec)**

Create your first specification document from scratch and learn the core workflow.

- Difficulty: Beginner
- Time: 15 minutes
- Prerequisites: SuperSpec installed

### Intermediate

**[Team Workflow](/tutorials/team-workflow)**

Learn how to use SuperSpec for collaborative team development.

- Difficulty: Intermediate
- Time: 30 minutes
- Prerequisites: Completed beginner tutorial

**[Complex Feature](/tutorials/complex-feature)**

Build a complete complex feature using Boost mode, experiencing the full SDD workflow.

- Difficulty: Advanced
- Time: 60 minutes
- Prerequisites: Completed beginner tutorial

## Learning Path

```
Beginner Stage
├── Install SuperSpec
├── Your First Spec
└── Learn basic commands

Intermediate Stage
├── Team Collaboration
├── Boost Mode
└── In-Contnet Learning

Advanced Stage
├── Complex Features
├── Custom Templates
└── CI/CD Integration
```
