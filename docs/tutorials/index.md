# 教程

通过实战教程快速掌握 SuperSpec 的使用方法。

## 教程列表

### 入门教程

<div class="tutorial-cards">

**[第一个 Spec](/tutorials/first-spec)**

从零开始创建你的第一个规格文档，了解 SuperSpec 的核心工作流程。

- 难度：初级
- 时间：15 分钟
- 前置：已安装 SuperSpec

</div>

### 进阶教程

<div class="tutorial-cards">

**[团队协作](/tutorials/team-workflow)**

学习如何在团队中使用 SuperSpec 进行协作开发。

- 难度：中级
- 时间：30 分钟
- 前置：完成入门教程

</div>

<div class="tutorial-cards">

使用增强模式开发一个完整的复杂功能，体验完整的 SDD 流程。

- 时间：60 分钟
- 前置：完成入门教程

</div>

## 学习路径

```
入门阶段
├── 安装 SuperSpec
├── 第一个 Spec（本教程）
└── 了解基本命令

进阶阶段
├── 团队协作
├── 增强模式
└── Vibe Coding

高级阶段
├── 复杂功能开发
├── 自定义模板
└── CI/CD 集成
```
