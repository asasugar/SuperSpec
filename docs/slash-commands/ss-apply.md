# /ss-apply

按顺序执行 tasks.md 中的任务。

## 语法

```
/ss-apply
```

## 参数

此命令没有参数。

## AI 行为

执行此命令后，AI 会：

1. 读取 tasks.md
2. 找到第一个未完成的任务
3. 执行任务（写代码、修改文件等）
4. 标记任务为完成 ✅
5. 继续下一个任务
6. 遇到问题时暂停并询问

## 执行过程

```
AI: 开始执行任务 1.1: 创建数据库模型
AI: [执行代码更改]
AI: ✅ 任务 1.1 完成

AI: 开始执行任务 1.2: 创建 API 路由
AI: [执行代码更改]
AI: ✅ 任务 1.2 完成

AI: 开始执行任务 2.1: 实现认证逻辑
...
```

## 中断处理

如果需要中断：

1. 当前任务会保持未完成状态
2. 使用 `/ss-resume` 恢复
3. AI 会从中断处继续

## 后续步骤

所有任务完成后：

```
/ss-archive [name]    # 归档变更
```

如果中断后需要继续：

```
/ss-resume    # 恢复上下文
```
